# 项目简介

## 核心功能
- 基于深度学习YOLOv8算法的电动车头盔佩戴检测
- PyQt5图形用户界面
- 完整源码和数据集提供
- 源码说明和配置跑通说明
- 配套报告

## 适用场景
- 交通管理部门对非机动车头盔佩戴情况进行监管
- 道路安全监控
- 道路交通事故预防

## 技术特点
- 高精度和实时性
- YOLOv8模型在小目标检测方面的优势
- 提升电动车头盔佩戴检测效率
